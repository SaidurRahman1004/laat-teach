HOW TO SEE CONNECTED WIFI PASSWOARD
WITHOUT ANY SOFTWOARE IN PC

In order to view the WiFi password on your Windows machine,
you need to make sure you are already connected to the WiFi network
or have previously connected to the WiFi network.


1. First, you need to open the "control panel". It can be opened by searching "Control Panel" from the search bar!

2. Open Control Panel and click on "Category".

3. Then you have to select the "large icon".

4. Select the "Network and Sharing Center" center option from the Control Panel.

5. Then click on the "WiFi option" from the connection.

6. Then the wifi Status dialog box will open. From there, click on the "Wireless Properties" option ...

7. Then click on the "Security" option from the WiFi-Wireless Network Properties dialog box.

8. In the security option "Network Security Key" this is the desired WiFi password.

9. From here, the password can be seen by clicking and ticking the "Show Characters" box.
